# Block client running apps based on hostname
Project ini didasarkan atas "ke-kepoan" Handy Kurniawan Ponco Widagdo wkwkwkw

## Fitur :
- Memblokir user untuk me-running aplikasi tertentu jika tidak ada di daftar allowed_pc_list.txt (misal aplikasi VIP tidak bisa dijalankan di REGULER)
- Auto hide isi folder jika mengaktifkan mode auto_hide_folder_after_blocked agar user tidak "nakal" (gunakan 'yes' atau 'no')
- Pesan warning dan nama exe yang bisa diatur sesuai keinginan

## Catatan :
- value dari auto_hide_folder_after_blocked hanya 'yes' dan 'no' dengan memperhatikan besar kecilnya huruf
- untuk mengganti icon exe, gunakan resource hacker
- gunakan Revert_Folder_hide.bat jika tidak sengaja membuka run_game.exe dalam mode auto_hide_folder_after_blocked=yes dan terbuka di pc yang tidak masuk dalam list pc allowed

## Cara Pakai :
Taruh di dalam folder game, lalu sesuaikan config.ini dan allowed_pc_list.txt sesuai dengan petunjuk, dan ubah shortcut game nya mengarah ke run_game.exe
